# Gushwork Frontend Assignment

## Project Overview
This project is a responsive web page developed using pure HTML, CSS, and JavaScript without any external frameworks or libraries. The implementation follows the provided Figma design specifications with focus on responsiveness, smooth interactions, accessibility, and clean code structure.

## Features
- Responsive design for desktop, tablet, and mobile devices
- Sticky header with smooth show/hide transition on scroll
- Interactive image carousel
- Hover zoom functionality for carousel images
- Semantic HTML5 structure
- Modern CSS using Flexbox and Grid
- Smooth animations and transitions
- Cross-browser compatible implementation
- Optimized and well-commented code

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
